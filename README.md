# 📊 Study Productivity Dashboard

An Excel-based dashboard designed to track daily study habits and analyze productivity trends.

## 🚀 Features

### Habit Tracking Dashboard

* Track daily progress for:

  * DSA
  * Aptitude
  * Projects
  * Python Practice
* Monthly habit monitoring
* Weekly completion analysis
* Done vs Remaining visualization
* Interactive donut charts

### Productivity Analytics Dashboard

* Daily productivity tracking
* Weekly productivity analysis
* Pivot Tables
* Slicers for filtering data
* Line, Pie, and Bar Charts
* Performance trend visualization

## 🛠️ Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Data Visualization

## 📈 Dashboard Components

### Habit Tracker

Tracks completion status of:

* DSA

### Analytics Dashboard

Provides insights into:

* Performance comparison
